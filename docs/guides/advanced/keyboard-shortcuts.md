# Keyboard Shortcuts

The BlockNote editor supports standard keyboard shortcuts for fast editing.

## Text Formatting

* **Ctrl+B** — Bold

* **Ctrl+I** — Italic

* **Ctrl+U** — Underline

## Block Operations

* **Enter** — Create a new block/paragraph

* **Backspace** (at start of block) — Merge with the previous block

* **Tab** — Indent a list item

* **Shift+Tab** — Outdent a list item

* **/** — Open the block type menu (paragraph, heading, list, code, etc.)

## Navigation

* **Ctrl+K** — Focus the search bar

* **Escape** — Close search results

* **Arrow Up/Down** — Navigate search results

*

* **Enter** — Select a search result
