# Lists and Tasks

Ordered lists, unordered lists, nesting, and interactive task lists.

## Ordered Lists

Start each line with a number and a period.

1. First item

2. Second item

3. Third item

```text
1. First item
2. Second item
3. Third item
```

## Unordered Lists

Start each line with a dash, asterisk, or plus sign.

* First item

* Second item

* Third item

```text
- First item
- Second item
- Third item
```

## Nested Lists

Indent with spaces or Tab to create nested items.

1. Main item

   * Sub-item one

   * Sub-item two

2. Another main item

   1. Numbered sub-item

   2. Another numbered sub-item

```text
1. Main item
   - Sub-item one
   - Sub-item two
2. Another main item
   1. Numbered sub-item
   2. Another numbered sub-item
```

Use **Tab** to indent a list item and **Shift+Tab** to outdent.

## Task Lists

Create interactive checklists with `- [ ]` (unchecked) and `- [x]` (checked).

* [x] Write the introduction
* [x] Add code examples
* [ ] Review and publish

```text
- [x] Write the introduction
- [x] Add code examples
- [ ] Review and publish
```

Checkboxes are clickable in the editor.
